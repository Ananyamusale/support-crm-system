# Support CRM System

A full-stack customer support ticket management system built using FastAPI, SQLite, HTML, Tailwind CSS, and JavaScript.

## 🚀 Features
- Create support tickets
- View all tickets
- Search tickets
- Filter by status
- Update ticket status and notes
- Delete tickets
- Dark mode UI
- Toast notifications
- Responsive dashboard

## 🛠 Tech Stack
- Backend: FastAPI  
- Database: SQLite  
- Frontend: HTML, Tailwind CSS, JavaScript  

## ▶️ Run Backend
uvicorn main:app --reload
